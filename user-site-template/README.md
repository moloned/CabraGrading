# Root GitHub Pages landing page template

Use this template in the repository named `moloned.github.io` to control:

- https://moloned.github.io/

## Steps

1. Create (or open) the GitHub repository named `moloned.github.io`.
2. Add the `index.html` from this folder to the root of that repository.
3. Commit and push to `main`.
4. In repository Settings -> Pages, set Source to `Deploy from a branch`, branch `main`, folder `/ (root)`.
5. Wait 1-2 minutes and reload https://moloned.github.io/.

If Pages is already set to GitHub Actions in that repository, switch to branch mode or add an Actions workflow there that publishes this `index.html`.
